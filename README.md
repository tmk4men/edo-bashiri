# 江戸走り 〜飛脚リズム〜

「江戸走り」をテーマにした、スマホ向けのシンプルなリズム・ミニゲームです。
HTML / CSS / Canvas の **1ファイル（`index.html`）** だけで動きます。インストール不要、ブラウザで開くだけ。

## 「江戸走り」とは？

61歳の研究家・大場克則さんが、江戸時代の文献や浮世絵をもとに **11年かけて再現** した走法。
2025年秋〜冬に TikTok・Instagram・YouTube ショートで **総再生2億7千万回超**、
**JC・JK流行語大賞2025で1位** を獲得しました。

**特徴**
- 体をやや横向き・前傾にして、**力を抜いて**走る
- 手足を内外に回す（**ナンバ走り** ＝ 同じ側の手足を同時に出す）
- **頭の高さを変えない** ので上下動が少ない
- 倒れる重力を推進力に使い、**力を抜くほど速くなる**

**メリット**
- 着地衝撃が小さく、膝への負担が少ない
- 脱力するので疲れにくく、長距離向き
- 筋力に頼らず、砂・泥・雪でも滑りにくい

> ※「江戸走り」という名称は歴史的な専門用語ではなく現代の呼び名、という指摘もあります。

## 遊び方

| 操作 | アクション |
|---|---|
| **タップ／クリック**（PCはスペースキー） | 一歩進む（リズム） |
| **長押し** | ジャンプ（障害物を飛び越える） |

- 足跡 👣 が足元の輪に重なった **瞬間にタップ** で一歩進む
- リズムが合うほど **コンボで加速**。テンポが乱れると **つんのめってコケる**
- 行く手の岩・桶・木箱は **長押しジャンプ** で回避
- バランスゲージが尽きるとゲームオーバー。走った距離（里）でランク判定

「あわてず、一定のリズムで」――江戸走りの極意がそのまま攻略のコツです。
